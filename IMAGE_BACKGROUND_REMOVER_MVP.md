# Image Background Remover - MVP产品文档

## 一、项目概述

**项目名称：** Image Background Remover（图片背景移除工具）

**核心定位：** 一款轻量级在线工具，用户上传图片即可自动移除背景，支持PNG透明背景下载。

**核心价值：** 操作简单、处理快速、成本最低化，面向有图片去背景需求的普通用户、电商卖家、设计师等。

**对标参考：** remove.bg（功能简洁、流程短）

---

## 二、用户故事与需求分析

### 2.1 核心用户场景

| 场景 | 用户描述 | 核心痛点 |
|------|---------|---------|
| 电商白底图 | 淘宝/京东卖家需要白底商品图 | 手动PS耗时、不会用PS |
| 证件照换背景 | 用户需要不同背景色的证件照 | 去打印店贵且麻烦 |
| 设计素材处理 | 设计师需要透明背景PNG素材 | 找图费时、抠图费事 |
| 快速换背景 | 运营人员需要快速给图片换背景 | 缺乏简便工具 |

### 2.2 用户故事（User Story）

- **作为**普通用户，**我希望**上传图片就能自动去掉背景，**这样**不用学习PS就能得到透明背景图。
- **作为**电商卖家，**我希望**5秒内得到去背景的商品图，**这样**我能快速上传到店铺。
- **作为**设计师，**我希望**下载的PNG保留透明通道，**这样**我可以直接用在设计稿里。

## 五、技术方案

### 5.1 技术栈选型

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 前端框架 | Vue 3 + Vite | 轻量、开发快、用户体验好 |
| UI组件 | 原生CSS / 轻量组件库 | 减少依赖、提升加载速度 |
| 后端 | Node.js + Express | 轻量、接API方便 |
| 图片存储 | 临时内存/本地存储 | 随用随删，不长期存储 |
| 图像处理API | **百度图像分割API** 或 **remove.bg API** | 有免费额度，精度高 |
| 服务器 | 轻量云服务器（1核1G） | 早期足够，后续可扩展 |
| 域名 | 按需注册 | 需备案（国内节点）或用海外节点免备 |
| HTTPS | 必须 | 浏览器安全要求 |

### 5.2 API选型对比

| API | 免费额度 | 单张成本 | 精度 | 集成难度 |
|-----|---------|---------|------|---------|
| 百度图像分割API | 500次/天 | 低（按量付费） | 高 | 中等 |
| remove.bg API | 50次/月（免费）| $0.05/张（付费）| 很高 | 简单 |
| 字节图像分割 | 按量付费 | 中 | 高 | 中等 |

**推荐方案：百度图像分割API（免费额度大，精度高，中国大陆访问快）**

## 七、部署方案

### 7.1 环境准备

| 资源 | 规格 | 预计成本 |
|------|------|---------|
| 云服务器 | 1核1G轻量应用服务器 | ¥200-400/年 |
| 域名 | .com 常规域名 | ¥50-80/年 |
| HTTPS证书 | Let's Encrypt（免费） | ¥0 |
| **合计** | - | **¥250-480/年** |

### 7.2 部署流程

```
1. 服务器初始化（Nginx + Node.js环境）
2. 前端构建（npm run build）
3. 前端静态文件部署到 /var/www/html
4. 后端服务部署（PM2守护进程）
5. Nginx 反向代理（API请求 → Node服务）
6. 域名解析 + HTTPS配置
7. 测试验证全流程
```

### 7.3 运维注意事项

- 定期清理 `/tmp` 目录下的临时图片（防止硬盘满）
- API调用频率限制（防滥用）
- 日志监控（Nginx access/error log）

## 十、验收标准

### 10.1 功能验收

| 编号 | 验收条件 | 测试方法 |
|------|---------|---------|
| UAT-01 | 上传JPG/PNG/WebP格式图片，正常显示预览 | 手动上传测试 |
| UAT-02 | 上传非支持格式，提示格式错误 | 上传GIF测试 |
